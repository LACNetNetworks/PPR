## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please report it via email to **security@lacnet.com**.

### When to Report

Please report security issues if:
- You find a vulnerability that can be exploited (XSS, CSRF, etc.).
- You find exposure of sensitive data.
- You find a breakage in authentication or authorization.

### How to Report

Please include the following details in your report:
- Description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Any proof of concept (PoC) code or screenshots.

### Response Time

Our security team will acknowledge receipt of your report within 48 hours and will provide an estimated timeline for addressing the issue.

### Disclosure Policy

We follow a [Coordinated Vulnerability Disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure) policy. We ask that you give us a reasonable amount of time to fix the issue before making it public.
